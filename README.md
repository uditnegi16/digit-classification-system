## Chapter 3 - Classification

### Topics Covered

- Binary Classification
- Confusion Matrix
- Precision
- Recall
- F1 Score
- Threshold Tuning
- Precision-Recall Tradeoff
- ROC Curve
- ROC-AUC
- Multiclass Classification
- One-vs-Rest (OvR)
- Error Analysis
- Multilabel Classification
- Multioutput Classification

### Key Learning Outcomes

- Accuracy can be misleading on imbalanced datasets.
- Threshold tuning directly affects precision and recall.
- ROC-AUC is useful for classifier comparison.
- Error analysis helps identify model weaknesses.
- Classification extends beyond binary problems into multiclass, multilabel, and multioutput tasks.
